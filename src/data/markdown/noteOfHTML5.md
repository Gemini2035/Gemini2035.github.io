# HTML5的新特性
***
## 语义化标签
### 1、header
定义文档的头部
### 2、nav
定义导航链接
### 3、section
定义文档中的节、段落
### 4、article
定义页面独立内容区域
### 5、aside
定义页面的侧边栏内容
### 6、details
定义文档某部分细节
### 7、summary
标签包含了details元素的标题
### 8、dialog
定义对话框、提示框
## Input表单新增
### 1、类型
tel number date time week month url email color rang
### 2、属性
required placeholder autofocus pattern multiple width height type=img autocomplete(展示历史输入) min max
## audio、video音视频相关
